## Variant: Command Library

### Design stance
A dense, editor/operator dashboard: search and triage first, decoration second.

### Key choices
- Layout: left navigation, central table, persistent right preview drawer.
- Typography: compact system UI with strong numeric/status hierarchy.
- Color: dark command-center palette with cyan pairing/active states, loosely informed by Refero-style product references.
- Interaction target: approve/open folder from the selected row without losing table context.

### Trade-offs
- Strong at: power-user review loops, local vault density, showing that this is a real tool.
- Weak at: first-run friendliness; may feel too pro/technical for a casual creator.

### Best for
Editors/operators reviewing hundreds or thousands of saved sounds.
