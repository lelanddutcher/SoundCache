set up the magic button 🪄

~2 minutes, zero coding, one (1) iPhone

  1. In the desktop app: Settings → Create pairing code, copy it. 📋
  2. On iPhone open Shortcuts → ＋ → add Receive URLs from Share Sheet.
  3. Add Get Contents of URL → POST to <your-relay>/v1/inbox/submit, JSON body { "pair_code": "YOUR-CODE", "url": Shortcut Input, "source": "ios_shortcut" }.
  4. Add Show Notification → "Sent to Sound Vault 💅". Name it Save to Sound Vault, enable in Share Sheet.
  5. Turn on auto-import on the desktop and watch your shares just... appear. ✨
